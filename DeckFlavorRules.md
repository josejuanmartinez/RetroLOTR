# RetroLOTR Modular Deck Design Rules

Use this as the canonical flavor guardrail when adding/moving cards.

## Global Rules (all factions)

1. **Base decks = reliability + identity core**
   - Generic economy, movement, utility, and foundational faction mechanics.
   - Avoid highly specific lore-title cards in Base unless mechanically essential.

2. **Flavor decks = sharper identity + narrower mechanics**
   - Cards should strongly match deck fantasy by name, effect, and tone.
   - Flavor cards can be swingier, but must have clear counterplay.

3. **No duplicate fantasy role across sibling flavor decks**
   - If one subdeck owns a motif, avoid leaking it into another unless intentional.

4. **Tag discipline**
   - New cards should include tags that match deck fantasy keywords.

5. **Distribution priority**
   - Name/lore specificity > mechanics specificity > generic utility.

## SharedBase

**Role:** neutral world layer (encounters + universal pressure)

**Allow:**
- Encounters, map hazards, neutral events, world-state shifts.

**Avoid:**
- Strong faction-exclusive signature effects.

## Gandalf Family

### GandalfBase
**Core fantasy:** versatile Free Peoples command + support.

**Allow:**
- General morale, scouting, movement support, diplomacy, balanced spell utility.

**Avoid:**
- Subdeck-specific titles/aliases unless card is staple core.

### GandalfTheGrey
**Fantasy:** wandering counselor, subtle setup, humble but clever plays.

**Preferred mechanics:**
- Reveal/scout, route utility, low-cost setup, protective tricks.
- Tempo-neutral value over brute finishers.

### GandalfTheWhite
**Fantasy:** returned authority, rally and righteous power spike.

**Preferred mechanics:**
- Teamwide courage/rally, anti-corruption cleansing, comeback stabilization.
- Leadership buffs and decisive turn swings.

### Mithrandir
**Fantasy:** arcane wisdom and focused wizardry.

**Preferred mechanics:**
- Precision spells, warding, insight/reveal, targeted control.
- High-skill magic with clean condition/effect structure.

### Stormcrow
**Fantasy:** omen-bringer, disruptive wanderer.

**Preferred mechanics:**
- Weather/event disruptions, reposition, unpredictability, soft denial.
- Tactical pivots and reactive plays.

### DisturberOfThePeace
**Fantasy:** troublesome adventure energy.

**Preferred mechanics:**
- Event-driven disruption, quirky utility, social/chaotic board disturbances.
- “Unexpected but fair” interaction.

### Tharkun
**Fantasy:** dwarf-friend and mountain alliances.

**Preferred mechanics:**
- Erebor/Khazad resonance, gear/treasure synergy, stubborn resilience.
- Defensive economy, fortified progression.

## Practical Move Rules (for future card distribution)

When deciding target deck for a card, score in this order:

1. **Lore-title match** (strongest): named identity/alias/location tied to one subdeck.
2. **Mechanic match**: card effect pattern matches subdeck mechanic profile.
3. **Tone match**: card mood/aesthetic aligns with subdeck fantasy.
4. **Fallback**: if unclear, keep in faction Base.

If 2+ subdecks tie, keep in Base and mark as `review-needed`.

---

## QA Checklist

Before finalizing distribution:

- Each subdeck has a clear role and no identity collision.
- Base decks still function alone (no critical mechanic missing).
- Flavor decks are additive, not mandatory for basic faction viability.
- No subdeck is overloaded with pure economy cards.
- High-swing cards are distributed across factions fairly.
